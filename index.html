Web Showcase: Integrasi API

Visualisasi UI berdasarkan data API simulasi.

1. Google Maps Directions API

Simulasi Booking Context menghitung jarak dan ETA untuk menentukan tarif.

Hasil Perhitungan Rute

Estimasi Jarak 5,3 km
Estimasi Waktu Tiba (ETA) 20 menit
Lihat Data Teknis (JSON)

Visualisasi Permintaan (Request)

GET https://maps.googleapis.com/maps/api/directions/json?origin=-6.9024,107.6186&destination=-6.9249,107.6366&key=YOUR_API_KEY

Visualisasi Respons (Response)

{
  "status": "OK",
  "routes": [ {
      "legs": [ {
          "distance": { "text": "5,3 km", "value": 5300 },
          "duration": { "text": "20 menit", "value": 1200 },
          "start_address": "Jl. Diponegoro...",
          "end_address": "Jl. Gatot Subroto..."
        } ]
    } ]
}

2. Payment Gateway API (Midtrans)

Simulasi Payment Context membuat tagihan (charge) untuk top-up Wallet.

Tagihan Top-Up Dibuat

Jumlah Tagihan Rp 100.000,00
Kode Virtual Account (BCA) 12345678901 Status: Menunggu Pembayaran
Lihat Data Teknis (JSON)

Visualisasi Permintaan (Request)

POST https://api.midtrans.com/v2/charge
Authorization: Basic (Base64-encoded YOUR_SERVER_KEY)
{
  "payment_type": "bank_transfer",
  "transaction_details": { "order_id": "order-topup-12345", "gross_amount": 100000 },
  "bank_transfer": { "bank": "bca" }
}

Visualisasi Respons (Response)

{
  "status_code": "201",
  "transaction_status": "pending",
  "order_id": "order-topup-12345",
  "gross_amount": "100000.00",
  "payment_type": "bank_transfer",
  "va_numbers": [ { "bank": "bca", "va_number": "12345678901" } ]
}
